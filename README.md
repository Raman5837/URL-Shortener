  - [Mongoose](https://http://mongoosejs.com)- MongoDB object modeling tool

## Getting Started

#### Clone the project

```sh
# clone it
git clone https://github.com/Raman5837/URL-Shortener.git
cd URL-Shortner
# Make it your own
rm -rf .git && git init
```

#### Run back end

```
# Move to server folder
cd server/
# Install dependencies
npm install

# Start  server
npm start
or
nodemon start dev
```
